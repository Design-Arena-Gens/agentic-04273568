A Next.js 14 site that explains the foundations of civics, highlights ways to participate in public life, and offers an interactive quiz to reinforce key ideas.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the experience locally.

## Available Scripts

- `npm run dev` &mdash; start a local development server
- `npm run build` &mdash; create an optimized production build
- `npm start` &mdash; serve the production build
- `npm run lint` &mdash; lint the project with ESLint
- `npm test` &mdash; alias for `npm run lint`

## Project Highlights

- **App Router:** modern Next.js structure with static rendering.
- **Accessible content:** semantic sections, focusable controls, and descriptive copy covering civics fundamentals.
- **Interactive learning:** expandable FAQ and a client-side quiz reinforce engagement concepts.
- **Deploy-ready:** optimized for deployment on Vercel as `agentic-04273568`.

## Deployment

```bash
npm run build
npm start
```

For production on Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-04273568
```
